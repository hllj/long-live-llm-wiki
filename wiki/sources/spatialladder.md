## Summary

SpatialLadder diagnoses the core failure of existing spatial reasoning approaches — they "treat spatial reasoning as a monolithic capability, attempting to learn it directly from question-answer pairs without establishing the necessary hierarchical structure" — and fixes it with a three-stage progressive training curriculum on a new multimodal dataset (SpatialLadder-26kk, 26,610 samples).

A controlled experiment confirms the diagnosis: adding bounding box hints to 200 spatial orientation tasks improves accuracy by **+5.0%**, and adding directional cues yields another **+4.5%**. The bottleneck is not reasoning capacity but perception grounding — models know *how* to reason spatially but cannot see *what* to reason about. This directly corroborates the projection bottleneck and frame-of-reference instability failure modes in [[concepts/spatial-reasoning-taxonomy]].

The resulting 3B-parameter model beats GPT-4o by 20.8% and Gemini-2.0-Flash by 10.1% on in-domain benchmarks, and achieves 45.7% on VSI-Bench — nearly matching the 4B Spatial-MLLM (47.3%) which uses specialised 3D encoders, using only standard VLM architecture.

## Key Points

### The Three-Stage Progressive Training Framework

Base model: **Qwen2.5-VL-3B**. Hardware: 4× NVIDIA A6000.

**Stage 1 — Perceptual Grounding (Localization)**
- Task: object localization from SpatialLadder-26kk (6,000 samples)
- Training: SFT (cross-entropy)
- Goal: distinguish spatially relevant objects from background; map language descriptions to visual regions
- *"Strong visual grounding before advancing to complex reasoning"*

**Stage 2 — Spatial Understanding (Multi-Dimensional)**
- Task: full spatial reasoning across single-image, multi-view (8 viewpoints), and video (1–4 min)
- Training: SFT across all three modalities
- Seven spatial dimensions: relative direction, relative distance, absolute distance, object size, counting, room size, appearance order
- Ablation: removing Stage 2 causes **−9.4% accuracy** — the largest single-stage contribution

**Stage 3 — Spatial Reasoning via GRPO**
- Training: Group Relative Policy Optimization with dual reward
  - Format reward: `<think>...</think><answer>...</answer>` structure required
  - Accuracy reward: exact match for multiple-choice; weighted relative accuracy across confidence thresholds [0.50–0.95] for numerical
- Cold start: 1,255 chain-of-thought samples generated by Qwen2.5-VL-7B via rejection sampling before GRPO
- Semantic entropy trajectory: Stages 1–2 raise entropy 1.24 → 1.47 (exploration); Stage 3 lowers 1.47 → 0.66 (convergence)

### SpatialLadder-26kk Dataset

| Task Category | Samples | Modality |
|---|---|---|
| Object Localization | 5,929 | Single-image |
| Single-image Spatial Reasoning | 5,929 | Single-image |
| Multi-view Spatial Reasoning | 5,752 | Multi-view (8 viewpoints) |
| Video Spatial Reasoning | 9,000 | Video |

Source: ScanNet 3D reconstructions (localization + single/multi-view) + SR-91k videos. ~90% of generated QA pairs filtered out via quality constraints (scene/object diversity, 40% visibility threshold, uniqueness).

### Results

**In-domain (proposed benchmarks):**

| Benchmark | SpatialLadder | Base (3B) | GPT-4o | Gemini-2.0-Flash |
|---|---|---|---|---|
| VSI-Bench | 45.7% | 29.4% | 34.0% | 45.4% |
| SPBench-SI | 70.2% | 40.3% | 42.4% | 54.7% |
| SPBench-MV | 70.9% | 36.6% | 48.2% | 51.4% |
| **Overall** | **62.3%** | **38.8%** | **41.5%** | **50.5%** |

**Out-of-domain generalization:**

| Benchmark | SpatialLadder | Base (3B) | GPT-4o |
|---|---|---|---|
| CV-Bench | 73.7% | 70.6% | 75.4% |
| SPAR-Bench | 34.4% | 24.6% | 36.4% |
| ViewSpatial-Bench | 44.2% | 35.6% | 32.6% |

Notable: **+16.5% on ViewSpatial-Bench person-perspective tasks** — robust transfer to novel egocentric viewpoints.

### Ablation Findings

| Component removed | Accuracy drop |
|---|---|
| Stage 2 (spatial understanding) | −9.4% |
| Single-image + multi-view data | −16.4% |
| Stage 3 (GRPO) | −2.1% |
| Stage 1 (localization) | −1.8% |
| Chain-of-thought | −0.8% |

Multi-view data diversity (−16.4%) and Stage 2 (−9.4%) are the two largest contributors. Stage 3 GRPO adds consistent but smaller gains.

### Attention Analysis

Spatial attention IoU on 400 SPBench-SI samples:

| Metric | SpatialLadder | Base |
|---|---|---|
| Task Accuracy | 73.5% | 32.1% |
| Visual Attention IoU | 37.7% | 33.8% |
| Attention Entropy | 0.176 | 0.193 |

Lower entropy + higher IoU = more concentrated attention on task-relevant spatial regions. Directly corroborates the attention geometry finding from [[sources/why-spatial-reasoning-hard-vlms]].
